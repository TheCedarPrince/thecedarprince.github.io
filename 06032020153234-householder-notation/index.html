Householder notation is very simple:

  1. Scalars are denoted as lowercase Greek letters: Î±\alphaÎ±. Î²\betaÎ², Î³\gammaÎ³, ...

  2. Vectors are represented as lowercase English letters: aaa, bbb, ccc, ...

  3. Matrices are denoted as uppercase English letters: AAA, BBB, CCC, ...

  4. Vector transposes are denoted with the apostrophe ( ' )

  5. Vector or matrix multiplication uses the star character ( * )
